1. Verificar el estado de un repositorio:
   git status

2. Agregar archivos individuales o globalmente.
   - Individuales:
     git add <filename>
     
   - Globalmente:
     git add .

3. Agregar comentarios al commit:
   git commit -m "Your commit message"
   

4. Cargar los cambios en el repositorio:
   git push origin <branch-name>
   
5. Crear, explorar y eliminar ramas:
   - Crear:
   git branch <new-branch-name>
   
   - Explorar:
     git checkout <branch-name>
          
   - Eliminar:
     git branch -d <branch-name>
    
6. Roll back de un repositorio  un commit
   git reset --hard <commit-hash>
